// OAuth 2.0 authorization code flow with JWT access tokens.

User [icon: monitor, color: sky]
Browser [icon: layout, color: indigo]
AuthServer [icon: shield, color: purple]
API [icon: server, color: coral]
DB [icon: table, color: teal]
Cache [icon: zap, color: amber]

User > Browser: click Sign In
Browser > AuthServer: GET /oauth/authorize
AuthServer > Browser: login form
User > Browser: enter credentials
Browser > AuthServer: POST /oauth/token
AuthServer > DB: lookup user by email
DB < AuthServer: user record + hashed password
AuthServer > AuthServer: verify password + sign JWT
Browser < AuthServer: access_token + refresh_token (httpOnly cookie)
Browser > API: GET /me (Authorization: Bearer token)
API > Cache: check token blacklist
Cache < API: not blacklisted
API > AuthServer: introspect token
AuthServer < API: claims { sub, roles, exp }
API > DB: SELECT profile WHERE id = sub
DB < API: profile row
Browser < API: 200 { name, email, avatar }
User < Browser: dashboard rendered
